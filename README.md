## **üî¨ Project Deep Dive: The Uniqueness Factor**

### **1. Federated Fine-tuning of LLMs with Private Data**
*A cutting-edge research implementation focused on privacy-preserving model training.*

- **Problem:** Fine-tuning LLMs is necessary for specialization, but client data is sensitive.
- **Solution:** Implemented **Federated Learning (using Flower framework)** combined with **Parameter-Efficient Fine-Tuning (PEFT)** on Llama2 models.
- **Achievement:** Enabled multiple clients to collaboratively train a shared LLM without ever exposing raw private data, demonstrating expertise in secure, distributed ML systems.
- **Technologies:** Flower, Llama2, PEFT, bitsandbytes (GPU optimization).

### **2. Multi-Modal Product Tagging Engine (Vision & Text)**
*High-performance system for e-commerce catalog automation.*

- **Architecture:** Developed a **Hybrid Dual-Encoder** model.
    - **Visual Stream:** **ResNet** for image feature extraction.
    - **Textual Stream:** **BERT** for processing product descriptions.
- **Impact:** Fusion layers accurately predict hierarchical tags, drastically improving **SEO, product discoverability**, and operational efficiency.
- **Technologies:** TensorFlow, PyTorch, ResNet, BERT, Multi-Modal AI.

### **3. LLM Based Question Generation Engine (Education)**
*Generative system leveraging LLMs for high-stakes content creation.*

- **Architecture:** Deployed **Gemini LLM** via **LangChain** with **Pydantic AI** for strict schema enforcement.
- **Impact:** Dynamically generates complex, semantically relevant **Question-Option pairs** for adaptive learning in JEE/NEET preparation.
- **Technologies:** Gemini, LangChain, Pydantic AI, NLP.
