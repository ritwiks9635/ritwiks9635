## **üöÄ Architectural Deep Dive: Flagship Projects**

My work focuses on leveraging **Agentic AI** and **Federated Learning** to solve complex, real-world problems while upholding data privacy and boosting operational metrics.

### **1. Federated Fine-tuning of LLMs with Private Data (Unique Project)**
*A cutting-edge research implementation focused on privacy-preserving model training.*

- **Problem:** Fine-tuning LLMs requires sensitive, proprietary data that cannot leave the client's environment.
- **Solution:** Implemented **Federated Learning (using Flower framework)** combined with **Parameter-Efficient Fine-Tuning (PEFT)** on Llama2 models.
- **Impact:** Enabled collaborative training of a high-performance, specialized LLM across multiple clients **without exposing raw private data**, establishing expertise in secure, distributed ML systems.
- **Technologies:** **Flower**, **Llama2**, **PEFT**, bitsandbytes, Distributed Computing.

### **2. Autonomous Hiring Agent & Trading Bot (Agentic AI)**
*Deployed intelligent, multi-step agents to automate high-value, complex workflows.*

- **Hiring Agent:** Built a **Pydantic AI**-driven system to automatically ingest and parse resumes, then use **Gemini LLM** and **LangChain/LangGraph** to dynamically generate tailored technical and behavioral interview questions.
- **Trading Bot:** Designed a **multi-agent financial assistant** using **LangGraph** to provide real-time stock insights, technical recommendations, and risk analysis via the **Gemini LLM**.
- **Technologies:** **LangGraph**, **Pydantic AI**, **Gemini**, Agentic AI, FastAPI.

### **3. Multi-Modal Product Tagging & Computer Vision**
*Bridging the gap between vision and language for automated asset management.*

- **Multi-Modal Engine:** Developed a **Hybrid Dual-Encoder** model using **ResNet** for image features and **BERT** for textual metadata, fusing them to predict hierarchical tags (category, color, style).
- **Pothole Segmentation:** Designed a real-time road video segmentation model using **YOLOv8** that achieved **93% detection accuracy** with fast inference for deployment.
- **Technologies:** **YOLOv8**, **ResNet**, **BERT**, Computer Vision, Multi-Modal Fusion.
